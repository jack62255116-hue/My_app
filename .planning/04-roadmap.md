## 스프린트별 상세

### M0 — 프로젝트 셋업 (05-18 ~ 05-25)

**목표:** 개발 환경 완성, 이후 스프린트에서 기능에만 집중

- [ ] Flutter 프로젝트 생성 (`flutter create myapp`)
- [ ] 패키지 설치: `sqflite`, `riverpod`, `dio`, `flutter_secure_storage`
- [ ] 폴더 구조 생성 (`core/`, `features/`, `shared/`)
- [ ] SQLite 초기화 및 `records` / `patterns` 테이블 마이그레이션
- [ ] CI: GitHub Actions Flutter 빌드 체크 설정
- [ ] 하단 탭바 Shell 화면 (빈 화면 4개)

**완료 기준:** 에뮬레이터에서 탭바 화면이 뜬다

---

### M1 — 수동 기록 (05-25 ~ 06-08)

**목표:** AI 없이도 기록·조회가 완전히 동작

**주차 1 (05-25 ~ 06-01)**
- [ ] 기록 입력 화면 UI (`+` 버튼 → 카테고리 선택 → 값 입력)
- [ ] SQLite CRUD (저장 / 수정 / 삭제)
- [ ] 홈 화면: 오늘 기록 목록 + 요약 카드

**주차 2 (06-01 ~ 06-08)**
- [ ] 히스토리 화면: 날짜별 기록 조회
- [ ] 항목 길게 누르기 → 수정 / 삭제
- [ ] 저장 후 스낵바 + 3초 취소 UX
- [ ] 기본 카테고리 5개 내장 (지출, 운동, 식단, 수면, 메모)

**완료 기준:** 기록 입력 → 저장 → 히스토리에서 확인까지 막힘 없이 동작

---

### M2 — AI 제안 (06-08 ~ 06-22)

**목표:** 핵심 차별점인 AI 자동 제안 동작

**주차 1 (06-08 ~ 06-15)**
- [ ] `PatternAnalyzer`: 최근 30일 데이터에서 시간대별 빈도 패턴 추출
- [ ] Claude API 클라이언트 구현 (`dio` 기반)
- [ ] API 키 `flutter_secure_storage`에 안전 저장
- [ ] 패턴 요약 → Claude API 호출 → 제안 파싱

**주차 2 (06-15 ~ 06-22)**
- [ ] AI 제안 카드 UI (수락 / 수정 / 건너뜀)
- [ ] + 버튼 탭 시 제안 우선 표시 플로우
- [ ] 건너뜀 피드백 → 패턴 가중치 감소
- [ ] 오프라인 시 제안 없이 수동 입력 폴백

**완료 기준:** 3일 이상 데이터 입력 후 AI 제안이 실제로 뜬다

---

### M3 — 완성도 & 베타 (06-22 ~ 07-06)

**목표:** 외부 사용자에게 보여줄 수 있는 수준

**주차 1 (06-22 ~ 06-29)**
- [ ] 온보딩 3장 슬라이드 (스킵 가능)
- [ ] 통계 화면: 주간·월간 막대 차트 (`fl_chart`)
- [ ] 카테고리 커스터마이징 (추가 / 삭제)
- [ ] 설정 화면: AI 제안 ON/OFF

**주차 2 (06-29 ~ 07-06)**
- [ ] 푸시 알림: 기록 안 한 경우 리마인더
- [ ] 앱 아이콘 / 스플래시 화면
- [ ] 빈 상태 UI (데이터 없을 때 안내 문구)
- [ ] 5명 지인 베타 테스트 + 피드백 수집

**완료 기준:** 지인 5명이 설치해서 하루 이상 사용

---
